## 🔧 常见问题快速解答

| 问题 | 答案 |
|------|------|
| 启动失败？ | 运行 `python launcher.py --check` 诊断问题 |
| 模型不存在？ | 确保 `bestvvv.pt` 和 `yolo11n.pt` 在项目根目录 |
| 摄像头无法打开？ | 检查系统权限和摄像头驱动 |
| 如何修改 Token 过期时间？ | 编辑 `config.py` 中的 `AUTH['token_expiry_hours']` |
| 如何添加新的检测模型？ | 在 `config.py` 中添加模型路径，扩展 `core/model_manager.py` |
